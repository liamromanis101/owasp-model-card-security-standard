# Changelog

All notable changes to the OWASP Model Card Security Standard will be documented here.

This project follows [Semantic Versioning](https://semver.org/). Given this is a documentation standard rather than software, version numbers reflect the maturity of the standard:

- **0.x** — Incubating; breaking changes possible between minor versions
- **1.0** — First stable release; breaking changes only in major versions thereafter

## How to read this changelog

Each release is documented with the following categories where relevant:

- **Added** — new sections, fields, or documents
- **Changed** — changes to existing content
- **Deprecated** — fields or sections that will be removed in a future version
- **Removed** — fields or sections that have been removed
- **Fixed** — corrections to errors or ambiguities
- **Security** — changes relating to the security properties of the standard itself
